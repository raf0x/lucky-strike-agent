# Lucky Strike Enterprise Retention Agent

AI-powered re-engagement tool for Lucky Strike Entertainment CSM team.
Select an at-risk enterprise account and generate a personalized re-engagement strategy in seconds.

Built with Next.js + Anthropic Claude API.

## Deploy to Vercel

1. Push this repo to GitHub
2. Import the repo at vercel.com/new
3. Add environment variable: `ANTHROPIC_API_KEY` = your key
4. Deploy

## Local development

```bash
npm install
# create .env.local with: ANTHROPIC_API_KEY=sk-ant-...
npm run dev
```
